Git adalah sistem kontrol versi distribusi yang digunakan untuk melacak perubahan dalam kode sumber perangkat lunak selama pengembangan proyek. Dengan Git, pengembang dapat membuat commit untuk merekam perubahan, membuat branch untuk mengembangkan fitur atau memperbaiki bug secara terisolasi, dan menggabungkan perubahan menggunakan pull request. Ini memfasilitasi kolaborasi tim, memungkinkan pengembang untuk bekerja secara efisien, dan menyimpan riwayat proyek dengan mudah. Repositori Git dapat berada secara lokal atau di-host secara eksternal, seperti di GitHub atau GitLab, memungkinkan akses dan kolaborasi yang lebih baik antar pengembang.
